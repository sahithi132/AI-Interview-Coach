# 📌 Overview

AI Interview Coach is an intelligent interview preparation platform designed to provide candidates with realistic interview practice.

The application leverages Google's Gemini AI to generate interview questions, evaluate user responses, identify strengths and weaknesses, and provide detailed improvement suggestions.

This project demonstrates full-stack development, AI integration, API development, prompt engineering, and modern software engineering practices.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/sahithi132/AI-Interview-Coach.git
```

---

## Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn main:app --reload
```

Backend Server

```
http://127.0.0.1:8000
```

---

## Frontend

```bash
cd frontend

npm install

npm start
```

---

# 💡 How It Works

1. User selects interview type.
2. Gemini AI generates interview questions.
3. User answers the questions.
4. AI evaluates responses.
5. Performance score is generated.
6. Personalized feedback is displayed.
